# Whist-Browser
Front end client

## Deploy

- Install wasm target: `rustup target add wasm32-unknown-unknown`
- Install trunk: `cargo install --locked trunk`
- Run `trunk serve`

## Development

### Requirements
#### Linux
The following packages are required to build locally.
```shell
sudo apt install libasound2-dev libudev-dev
```
